# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its Code of Conduct. The full, authoritative text is published at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Scope

The Code of Conduct applies to all project spaces — the repository, issue tracker, pull requests, discussions, and any public communication channel linked from this project. It also applies when an individual is officially representing the project in public spaces.

## Reporting

To report a concern, email **conduct@contriwork.dev**. Reports are reviewed by the maintainers listed in [`CODEOWNERS`](./.github/CODEOWNERS). We aim to acknowledge within five to seven business days.

Security vulnerabilities must **not** be reported here — see [`SECURITY.md`](./SECURITY.md) instead.

## Enforcement

Maintainers follow the **Community Impact Guidelines** published with the Contributor Covenant 2.1 (see the upstream link above) when determining responses, up to and including a permanent ban from all project spaces.
